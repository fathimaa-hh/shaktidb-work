# CodeQuest

Coding Progress and Skill Analytics Platform using ShaktiDB

## Overview

CodeQuest is a web application that helps students track coding practice, monitor contest performance, and analyze problem-solving progress through interactive dashboards.

Frontend
- React.js
- HTML
- CSS
- JavaScript

Backend
